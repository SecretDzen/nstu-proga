РГР по Программированию
Группа: АВТ-019
Вариант: 17
Задача: Заменить все символы с кодом больше 48 на пробел

Для передачи данных используются события. Файлы отображаются на память
Логи пишутся в log.txt
Обработанные файлы получают название <filename>.out

Сборка:
make all

Для управления сервером (myserv.service) используется программа systemctl с соответствующими опциями (start, stop, restart, status)

Клиент:
./client <count> <filename[, ...]>
